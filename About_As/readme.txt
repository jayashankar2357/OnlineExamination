Hello and thank you for using 3D Carousel Menu AS3 FX. By downloading our FlashXML product you have become part of the FX community and we are very pleased to welcome you and wish you a long-lasting FlashXML experience.

The 3D Carousel Menu AS3 FX's features are:
	- resizable to the extent of 1680 x 1050 pixels- rotation in any angle- optional centered image- customizable images width & height- all possible rotations behaviors & speeds- multiple roll over effects- HTML formatted tooltip- optional image frames & reflection

This is a list with all the product's attributes and a short description about what each attribute does:
componentWidth - carousel's width
componentHeight - carousel's height
preloader - loading animation
skin - carousel's controls set
radiusX - carousel's horizontal radius
radiusY - carousel's vertical radius
centerX - carousel's horizontal position
centerY - carousel's vertical position
perspective - carousel back images' size
rotationAngle - carousel's rotation angle
rotatioAngleType - carousel rotation: path rotates only image, all rotates all elements
useAlphaEffect - fade effect for carousel images
alphaIntensity - carousel's fade position
showCenterImage - show/hide carousel's center image
centerImageAlpha - carousel's central image opacity
centerImage - carousel's central image
centerImageUrl - carousel's central image hyperlink
centerImageUrlTarget - open central image link in the same window(_self) or in a new tab(_blank)
useColorbox - use image pop-up effect on click
rotateOnRollOver - how to rotate the carousel's images with your mouse
rollOutSpeed - carousel's rotation speed
rollOverSpeed - carousel's rotation speed on mouse over
inverseRotation - switch the carousel's rotation direction
thumbWidth - images' width
thumbHeight - images' height
borderCornerRadius - images' corners adjustment
borderWidth - images' border size
borderAlpha - images' border opacity
borderColor - text field's border color
useReflection - use images' reflection
reflectionAlpha - reflection opacity
reflectionSpread - reflection size
reflectionDistance - reflection distance from images
effectOut - images' brightness effect on mouse out
effectOver - images' brightness effect on mouse over
effectRatio - images' brightness effect speed
effectType - images' brightness effect on mouse over
visibility - text's visibility
position - text's position
delay - text's appearance delay
distance - text's distance from image
padding - text's padding
cornerRadius - text's corners adjustment
appearance - text's appearance direction
easingFunction - text's appearance animation
easingDuration - text's appearance animation duration
fontFace - text's font
fontSize - text's font size
fontColor - text's font color
backColor - text field's background color
backColor2 - text field's gradient color
backAlpha - text field's background opacity
borderThickness - text field's border size
useArrow - use arrow for the text field
arrowSize - text field's arrow size
embedFonts - use text smoothness
fillType - background type
backgroundImage - background image
backgroundFitting - background fitting
backgroundColor1 - background color when image background type not used
backgroundColor2 - background color when linear or radial background type used
backgroundAlpha - background opacity
bgCornerRadius - background corners' adjustments
gradientRotation - background colors rotation
gradientRatio - background colors adjustment
gradientPosX - background colors' horizontal position
gradientPosY - background colors' vertical position
useShade - use over mask
shadeColor - mask color
shadeAlpha - mask opacity
shadeRatio - mask adjustment
shadeWidth - mask width
shadeHeight - mask height
